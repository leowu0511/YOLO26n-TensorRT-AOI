# 🚀 2026 智慧創新大賞：高效能 AOI 瑕疵檢測引擎

### 硬體加速與底層優化實踐 (Target: NVIDIA RTX 4060)

## 📌 核心價值主張 (Value Proposition)

本專案開發了一套針對 PCB 電子零件瑕疵偵測的高效能推論引擎。透過從 Python 研究環境遷移至 **C++ 底層開發**，並深度整合 **NVIDIA TensorRT 10.15** 加速技術，成功將推論效能推向硬體極限，滿足現代工業產線對於「高精度、零延遲」的嚴苛要求。

---

## 🛠️ 技術突破亮點 (Technical Highlights)

* **極致效能優化**：捨棄傳統 Python 封裝，採用 C++ 進行全手動記憶體管理，消除 GIL 限制，達成單機 **500+ FPS** 的算力吞吐。
* **零拷貝技術 (Zero-Copy Mode)**：實作顯存預分配與 Pinned Memory 技術，大幅降低 PCIe 數據傳輸開銷，將端到端延遲壓低至 **1.9ms**。
* **GPU 預處理流水線**：將影像縮放 (Resize) 與歸一化 (Normalization) 由 CPU 移交至 GPU 執行，釋放 CPU 資源，確保在高吞吐量下系統依然穩定。
* **工業級穩定度**：針對 RTX 4060 硬體架構進行最佳化，P95 延遲波動小於 **1ms**，確保在高速產線上不會發生漏檢。
